## 说明
一个Dapp的开发、测试、发布的完整案例

## 结构说明
- build 编译文件
- contracts 合约文件
- migrations 部署文件
- node_modUles 依赖
- src 前端源文件

     - css 布局文件
     - fonts 字体文件
     - images 图片文件
     - js 页面逻辑文件

        - app.js 初始化与其他调用逻辑
        - bootstrap.min.js 
        - truffle-contract.js 合约相关
        - web3.min.js   
     - index.html 主页
     - pets.json 业务相关（宠物的属性文件）

- test 测试相关文件夹
- truffle.js truffle的配置文件（包含网络配置）
- package.json 包配置
